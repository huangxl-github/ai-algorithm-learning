# 项目概述

## 项目名称

人工智能算法学习

## 核心愿景

为学习者提供一条从数学与编程基础，到经典机器学习、深度学习、前沿 LLM/Agent 技术，再到工程化落地的**系统化、可执行**的人工智能算法学习路径。

## 主要目标

1. 整理并评估 `docs/人工智能算法学习路线图.pdf` 中的学习规划
2. 输出结构化的 `readme.md` 作为主线学习指南
3. 根据学习者背景（零基础 / 有基础 / 在职转行）提供时间调整建议

## 解决的问题

- 原 PDF 路线内容全面但时间表述易产生误解
- 部分知识点顺序（如 Transformer、强化学习实践）可优化
- 缺少分阶段验收标准与个性化调整说明
